# 项目简介

## 核心功能
- 系统分为管理员、护理师、用户三个角色
- 网站首页包含首页、宠物信息、公告信息、智能客服、个人中心等内容
- 管理员可对系统首页、个人中心、用户管理、护理类型管理、宠物分类管理、宠物护理管理、交流论坛、系统管理进行操作管理

## 适用场景
- 宠物行业企业
- 宠物护理机构
- 宠物爱好者社区

## 技术特点
- 架构：B/S、MVC、Web框架
- 开发环境：IDEA、JDK1.8、Mysql、Maven、Navcat
- 技术栈：Java、SpringBoot、Vue
